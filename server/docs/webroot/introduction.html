  • Explanation of Wolfpack

    Wolfpack is an Ultima Online server emulator. There is a long history of different Ultima Online emulators, many being branches of each other and all with their own defining characteristics. Wolfpack is another one of these emulators, and we have chosen a path for the server that gives it a great edge in functionality when compared to other emulators.

    Wolfpack now uses Python and XML as a script and definition base. It also uses the QT library for threading and other functions.

  • Explanation of Ultima Online

    Ultima Online is an old, yet still very popular MMOG. It is a game set in a fantasy environment, played through an isometric viewpoint. For more information please visit http://www.uo.com/.

  • Getting Started

    To get started, you will need to have Ultima Online installed. This can be done by going to your local store and buying it; Usually $30 USD or by finding one of their free versions online. The newer your version the better your functionaly will be.

    After you have Ultima Online installed, you will need to download the latest version of Wolfpack. This can be done by visiting the project files page found: here. Please note that the world files are for an ancient version of Wolfpack and will NOT work with the latest versions.

  • Managing Your Items and NPCs

    The base directory where definition files are stored is in ./definitions/ . There should be in index.xml file in almost every subfolder within. You can edit these files to modify which definitions are loaded into Wolfpack.

  • Managing Your Scripts

    The default file where Python scripts are defined can be found at definitions/scripts.xml . This file follows the following format:

    <definitions>
      <script>directiory.scriptname</script>
    </definitions>
